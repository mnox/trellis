# Contributing to Trellis

Thanks for your interest in contributing!

## Getting started

### Prerequisites

- Node.js 18+
- A [Convex](https://convex.dev) account (free tier works)

### Running locally

```bash
npm install
npm run dev   # starts convex dev — opens the Convex dashboard and watches for changes
```

## How to contribute

1. Fork the repo and create a branch from `main`
2. Make your changes
3. Make sure everything works locally with `npm run dev`
4. Open a pull request

## Pull request guidelines

- Keep PRs focused — one thing at a time
- Describe what you changed and why in the PR description
- If you're touching the schema or taxonomy, explain the intent
